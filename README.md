# 🤖 AI-Powered Feedback Analysis Tool

An AI-powered web application that analyzes user feedback and automatically determines the **sentiment** of comments using **Natural Language Processing (NLP)**. The system classifies feedback as **Positive**, **Negative**, or **Neutral**, enabling organizations to gain actionable insights from customer reviews, surveys, and user comments.

## 🚀 Features

- 🤖 AI-powered sentiment analysis
- 💬 Automatic classification of feedback into Positive, Negative, or Neutral
- 📊 Real-time feedback processing
- 📈 Interactive dashboard for sentiment visualization
- 🔍 Instant analysis of customer reviews and comments
- 📱 Responsive and user-friendly interface
- ⚡ Fast and accurate sentiment prediction

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/AI-Powered-Feedback-Analysis-Tool.git
```

Navigate to the project directory:

```bash
cd AI-Powered-Feedback-Analysis-Tool
```

Install frontend dependencies:

```bash
npm install
```

Install backend dependencies:

```bash
pip install -r requirements.txt
```

## ▶️ Running the Project

Start the frontend:

```bash
npm run dev
```

Start the backend:

```bash
python app.py
```

The application will be available at:

```
Frontend: http://localhost:5173
Backend: http://localhost:5000
```

## 🧠 How It Works

1. Users enter customer feedback or comments.
2. The text is preprocessed using NLP techniques.
3. The trained Machine Learning model analyzes the sentiment.
4. The application classifies the feedback as:
   - 😊 Positive
   - 😐 Neutral
   - 😞 Negative
5. Results are displayed instantly with sentiment insights.
